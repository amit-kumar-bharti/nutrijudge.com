Return Policy

Last updated: January 15, 2025

Return Window

You have 30 days from the date of delivery to return most items for a full refund or exchange.

Return Conditions

To be eligible for return, items must be:

  • In original, unopened packaging
  • Not used or damaged
  • In resalable condition
  • Returned with all original accessories and documentation

Non-Returnable Items

The following items cannot be returned:

  • Personal care items (for hygiene reasons)
  • Perishable goods
  • Items marked as "Final Sale"
  • Custom or personalized items

Return Process

  1. Log into your account and go to "Orders"
  2. Select the order you want to return
  3. Choose the items and reason for return
  4. Print the return label or schedule pickup
  5. Package the items securely
  6. Ship or hand over to our courier partner

Return Shipping

Return shipping is free for items returned due to our error or defective products. For other returns, shipping costs are the responsibility of the customer.
